# zsf-blog

## 1. 项目介绍
项目描述：为用户提供写作、分享与交流，为管理者提供管理功能的博客平台  
技术选用：Spring+Spring MVC+MyBatis+MySQL+JSP  
主要实现功能：  
1. 用户模块：登录、退出、注册、修改个人信息
2. 博客模块：编辑、修改、查询、删除、点赞、收藏博客
3. 管理员模块：文章管理、用户管理、栏目管理、数据统计

## 2. Bug

3. 发表博客时未作非空校验，导致外键为空的BUG
